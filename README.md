# ESTE CONTATO NÃO ESTÁ MAIS DISPONÍVEL

> *Um mensageiro sobre memória, ausência e sistemas que ainda observam.*

---

# Sobre a obra

“Este contato não está mais disponível” não é um erro técnico.  
É uma constatação existencial.

Esta obra é uma instalação digital interativa inspirada na interface do antigo MSN Messenger (Windows 2000/XP), subvertida pela estética Weirdcore / Liminal Space.

Aqui, a comunicação não visa eficiência.  
Ela existe para evocar.

O sistema responde.  
A interface lembra.  
O silêncio também é uma mensagem.

---

# Conceito

A obra simula um ambiente familiar, porém deslocado no tempo e na função.  
Texturas degradadas, filtros visuais instáveis, mensagens fantasma e sons imperfeitos criam uma sensação de nostalgia desconfortável — uma internet que prometia presença constante, mas deixou apenas vestígios.

O usuário não é exatamente um usuário.  
É um visitante.

Este não é um chat social.  
É um resíduo.

---

# Experiência

- Interface inspirada no MSN Messenger 6.2
- Estética weirdcore / dreamcore
- Mensagens automáticas do “sistema”
- Distorção visual e tipográfica
- Efeito Nudge (trepidação)
- Texto curatorial incorporado à interface

Cada sessão é levemente diferente.
